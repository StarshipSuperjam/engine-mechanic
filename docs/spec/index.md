# Product spec

not leaves: each document states the invariants, walls, and contracts a finished build must keep,
one short paragraph per law, citing the decision records in `../adr/` for the why and never
restating it. The authoring rules live in [principles](../principles.md); the shape of the whole in
[architecture](../architecture.md).
